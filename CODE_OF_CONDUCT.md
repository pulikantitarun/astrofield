# Code of Conduct

AstroField contributors and maintainers are expected to create a respectful, welcoming and harassment-free community.

Be constructive, assume good intent, respect differing experience levels, and focus criticism on technical work. Harassment, discrimination, threats, doxxing, sexualized conduct and sustained disruption are unacceptable.

Maintainers may edit or remove contributions and restrict participation when behavior is unsafe or harmful. Report conduct concerns privately to the repository owner through their GitHub profile. Reports will be handled as confidentially as practical.

This policy applies in repository discussions, issue trackers, pull requests and project-related public spaces.
