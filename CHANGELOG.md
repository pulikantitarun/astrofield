# Changelog

All notable changes will be documented here.

## Unreleased

### Added

- Landscape Flutter application for Android and iOS.
- Astroberry Python bridge with token-authenticated local API.
- Rig, Sky, Camera, Focus, Guide, Gear, Polar, System and Session workspaces.
- Phone-to-Pi location synchronization.
- Equipment profiles, INDI discovery, PHD2 Guiding Assistant and autofocus orchestration.
- Pi FITS storage, Samba sharing, Avahi discovery and system status foundations.

### Known limitations

- Early prototype; not all controls have completed real-hardware validation.
- iOS has not yet been tested on a physical device.
- Secure per-installation pairing and internet relay are not implemented.
- Sky framing, mosaics, polar alignment and session automation require further end-to-end validation.
